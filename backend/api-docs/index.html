MediaListItem - Up

Media asset info to show in lists
id (optional)
BigDecimal Media ID format: int64
type (optional)
BigDecimal

Media Type:

  • 0 - Deleted media
  • 1 - Image
  • 2 - Video
  • 3 - Audio
format: int32
title (optional)
String Media title
example: Media title
thumbnail (optional)
String Path to reach the thumbnail. Empty if no thumbnail. Thumnails are squared images.
example: /assets/b/0/0/thumbnail.jpg
duration (optional)
Double Duration of the media in seconds (videos and audios) format: double
width (optional)
BigDecimal Image/Video width format: int32
height (optional)
BigDecimal Image/Video height format: int32

TaskInfo - Up

Task status info
id (optional)
BigDecimal Task ID format: int64
running (optional)
Boolean True if the task is running
media_id (optional)
BigDecimal Media ID format: int64
type (optional)
BigDecimal

Task Type:

  • 0 - Encode original
  • 1 - Encode extra resolution
  • 2 - Generate video previews
format: int32
resolution (optional)
Object Resolution info
stage (optional)
String

Task Stage:

  • PREPARE - Preparing task resources
  • COPY - Decrypting and copying media assets
  • PROBE - Extracting metadata from media assets
  • ENCODE - Encoding
  • ENCRYPT - Encrypting new assets
  • UPDATE - Updating vault
  • FINISH - Finishing (clearing temp files, etc)
Enum:
PREPARE
COPY
PROBE
ENCODE
ENCRYPT
UPDATE
FINISH
stage_start (optional)
BigDecimal Stage start timestamp (Unix millis) format: int64
stage_progress (optional)
Double Stage progress: A number from 0 to 100 format: double

UserConfig - Up

User configuration
max_tasks (optional)
BigDecimal Max number of tasks in parallel. Set to 0 for no limit. format: int32
encoding_threads (optional)
BigDecimal Max number of threads to use for FFmpeg tasks. format: int32
resolutions (optional)
array[Object] Extra resolutions to encode videos
image_resolutions (optional)
array[Object] Extra resolutions to encode images
pinned_tags (optional)
array[Object] Pinned tags by the user
