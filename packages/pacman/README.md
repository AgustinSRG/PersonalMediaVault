# Pacman package builder

This is a script to build the pacman package in order to install the project in Arch Linux.

## Requirements

- [Golang](https://go.dev/) - Last stable version
- [NodeJS](https://nodejs.org/) - Last stable version
- [Rust](https://www.rust-lang.org/) - Last stable version

## Building

Crate the `pkg.tar` package file with the following command:

```sh
./build.sh
```
